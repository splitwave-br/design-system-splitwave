# Filter

O componente \`Filter\` é usado para criar filtros dinâmicos e personalizáveis com campos e opções variadas. Ele permite a filtragem de dados através de campos de entrada, seleções e botões.

### Uso do Hook \`useFilter\`:

Para registrar e gerenciar o estado do filtro, você pode utilizar o hook \`useFilter\`:

```tsx
const filterRegister = useFilter({ ...config });
const { ... } = filterRegister; // Para acessar as funções e estados do filtro
```

Você também pode usar o useFilterContext se estiver dentro do <Filter.Wrapper />:

```tsx
const { ... } = useFilterContext(); // Disponível somente dentro de <Filter.Wrapper />
```

Esses hooks permitem que você acesse e controle o estado do filtro de maneira centralizada.
