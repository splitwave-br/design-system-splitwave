# Table

O componente `Table` é usado para criar tabelas personalizáveis.

```tsx
    import { Table, Header, Cell } from "design-system/Table"

    function Component() {
        return (
            <Table<TData>
                data={[
                    { id: "1", name: "Lucas Costa Amaral", email: "lucas@splitwave.com.br" },
                    { id: "2", name: "Thiago Nunes", email: "thiago@splitwave.com.br" },
                    { id: "3", name: "Eduardo Souto", email: "eduardo@splitwave.com.br" },
                    { id: "4", name: "Erika Nishimura", email: "erika@splitwave.com.br" },
                ]}
                renderHeader={() => (
                    <>
                        <Header width="80px">#</Header>
                        <Header>Nome</Header>
                        <Header>E-mail</Header>
                    </>
                )}
                renderRow={({ date, name, price, status }) => (
                    <>
                        <Cell.Text>{item.id}</Cell.Text>
                        <Cell.Text>{item.name}</Cell.Text>
                        <Cell.Text>{item.email}</Cell.Text>
                    </>
                )}
            />
        )
    }
```
